================================ Outils ================================

* exporteurs 3DSMAX : 
  * exporteur de modeles : generation de fichiers decrivant la modelisation 3D des objets
  * exporteur de squeletes : generation de fichiers decrivant le squelette des objets, utilise pour l'animation skelettique
  * exporteur de materiaux : generation de fichiers decrivant un materiaux (reflexion, texture, ...)
  * exporteur d'animations : generation de fichiers decrivant les animations squelettiques des objets
  * exporteur de physique :  generation de fichiers decrivant l'interaction des objets avec leur environnement physique (collision, pesanteur, ...)

* editeur de niveau : generation et modification des niveau de jeux conenant les objets place dans leur environnement ainsi que toutes les informations necessaire a la creation d'un niveau de facon generale
  * implenmentation du systeme de configuration pour l'editeur et le lanceur (Fox-Toolkit)

================================ Pavillon-Noir le jeu ================================

* lanceur de Pavillon-Noir : permet de regler la configuration de l'ensembles des modules avant de lancer le jeux proprement dit (plein ecran, definition, volume, ...)

* module commun : base du systeme partage enre tous les modules
  * gestionnaire d'evennements : piece maitresse du systeme, il est charge de gerer le reseau de communication entre les modules
  * systeme de configuration : 
  * gestionnaire de modules : charge et initialise l'ensemble des modules
  * systeme d'internationnalisation : permet de gerer de facon simple au niveau du code les differentes langues possibles

* module rendu 3D : 
  * boite d'horizon : 
  * animation squelettique : permet de rendre des animations de personnages realistes
  * rendu 3D optimise avec textures : couche basse du module graphique il surcharge OpenGL et gere le rendu de mesh complexes et le dialogue avec la carte graphique
